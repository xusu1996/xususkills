# xususkills

Personal Codex skills for publishing reusable AI workflows.

## Skills

- `dialogue-screenshot-cards`: Turn real human-AI conversation excerpts into clean vertical screenshot-style social cards while preserving original wording.
- `thinking-evolution-journal`: Analyze Chinese diary entries into concise reports about emotions, causal chains, growth stages, and long-term patterns.

## Install

After this repository is published to GitHub, install a skill from its GitHub path:

```text
https://github.com/xusu1996/xususkills/tree/main/skills/dialogue-screenshot-cards
```

```text
https://github.com/xusu1996/xususkills/tree/main/skills/thinking-evolution-journal
```

Restart Codex after installing new skills.

## Notes

Each skill is self-contained in its own folder under `skills/`.
The required entry point is `SKILL.md`; `agents/openai.yaml` provides UI-facing metadata such as display name, short description, and default prompt.
